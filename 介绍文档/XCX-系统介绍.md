# 山药铭记小程序 - 系统介绍

## 1. 系统概述

“山药铭记”是一款基于微信小程序开发的专注中医学习与背诵的效率应用。应用内置了丰富的中医经典典籍（包括《伤寒论》、《金匮要略》、《温病条辨》、《黄帝内经》）、方剂库以及中药库。用户可以通过系统提供的“随时背”功能高效记忆各类条文及医学知识，还可以通过学习打卡记录自身学习进度，并在交流社区分享心得。系统采用前后端分离架构，实现用户端体验和后台管理的完美结合。

### 核心特点
- **专业且丰富的中医数据库**：内置多部核心中医经典以及各类方剂和中药典籍，经过精心归类与层级划分。
- **随时背机制**：支持针对特定条文、方剂的专属背诵和学习模式，快速巩固。
- **互动性与激励体系**：具备学习进度追踪功能，并提供点赞、积分商城机制支撑。
- **社区与知识反馈系统**：支持用户讨论交流和对医书文字排版的错漏意见反馈。

## 2. 核心功能模块

### 2.1 学习与背诵系统 (随时背)
- **核心经方条文背诵**：支持多级分类索引，针对不同级别的条文（例如一级必背条文与二级选读条文）进行快速背诵。
- **内容收藏与学习记录**：标记个人学习进度，一键保存重点、难点条文。

### 2.2 用户与激励机制
- **积分系统与商城（Shop）**：用户通过打卡、学习获得学习积分，积分可用于兑换商城中的实体周边或高级电子学习资料。
- **用户邀请与绑定**：提供邀请码功能（Invite Code），支持通过师兄/同学之间的邀请奖励绑定系统。

### 2.3 资料与社区互助
- **专区资料库（Materials）**：包含进阶中医学习资料库，支持阅读功能。
- **学习平台反馈（Feedback）**：提供专门错漏字纠错入口（针对古籍的即时修订提报），以及中医同仁之间的内容探讨区。

### 2.4 其他功能
- **个人总览（Profile）**：查看签到记录、学习足迹（History）、收藏夹展示（Favorites）。
- **滚动公告栏（Notices）**：实时获取后台下发的站内系统通告或功能上新介绍。

## 3. 技术环境
- **客户端架构**：微信小程序原生开发方案（WXML + WXSS + JS）。
- **数据状态流**：微信小程序 Storage 缓存结合远端数据库数据同步。
- **网络通信**：标准的 `wx.request`，统一调用 RESTful API。

## 5. 开发建议与后续扩展维护
- 因为系统主打中医古籍整理阅读，请必须注意部分罕见的**中医繁体生僻字、异体字**在小程序组件渲染层上的适配兼容度。
- 考虑到《伤寒论》等部分经典内容在实际展示中有上下层级嵌套庞杂情况（如“条文-注-附方”），小程序应在长文展示中使用更为扁平明朗的视图或支持划线折叠区域机制。
- 用户的 `Progress` 与 `History` 非常关键，务必定期执行远端云同步确保账号切换时的数据安全无遗漏。
